# Credit Architect

A modern and elegant payment plan calculator for loans and credits.

## Features
- **Dynamic Calculation**: Real-time updates as you adjust values.
- **Visual Table**: Clear, sticky-header table for payment schedules.
- **Responsive Design**: Works great on desktop and mobile.
- **Multi-language**: Support for Turkish and English.

## Tech Stack
- React + TypeScript
- Vite
- CSS Grid & Claymorphism

## Local Development
```bash
npm install
npm run dev
```
